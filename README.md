# tagTree
树型标签选择器

该控件是基于jquery库开发的，所以使用前，必需引入这些库。
同时选择勾勾为字体库图标，如果不想引入图标字体库，可以自己改成图片。
该控件是一个树型选择器，支持多选单选，并返回值。
